# STM32CAR

Little toy car base on STM32 NUCLEO F103RB driven by mobile app using accelerometer.

## Technologies

* Xamarin Forms
* UrhoSharp Library
* ACR Reactive BluetoothLE Plugin
* STM32 Nucleo F103RB
